# Images Folder

Place your placeholder fragrance bottle image here.

## Recommended file:
- `placeholder.jpg` or `placeholder.png`
- Dimensions: 400x500px (3:4 aspect ratio)
- File size: < 200KB

## Usage in code:
Reference as `/images/placeholder.jpg` in your image URL field.

The default placeholder is currently set to use Unsplash, but you can replace it with your own image by placing it here.
